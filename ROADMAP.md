# Noteriv Roadmap

Planned features for upcoming releases.

## Planned Features

- [ ] **Extract Selection to New Note** — Select text in the editor, right-click "Extract to new note." The selection moves to a new file and is replaced with a `[[wiki-link]]` back to it. Ideal for refactoring long notes into atomic, linked notes.

- [ ] **Advanced Search Filters** — Extend vault search with structured filters: `tag:#project`, `folder:journal/`, `created:last-week`, `has:tasks`. Combine multiple filters for powerful queries beyond plain text search.

- [ ] **Note Encryption (per-note)** — Lock individual notes with a password. Content is encrypted on disk (AES-256) while the filename stays visible. Unlock inline when editing. Privacy-first, local-only encryption.

- [ ] **Export to DOCX/EPUB** — Expand export beyond PDF with Word (.docx) and eBook (.epub) formats. Useful for sharing with non-technical collaborators or self-publishing workflows.

- [ ] **Clipboard History** — A panel showing recently copied/cut text snippets within the app. Click any previous clip to paste it. Cleared on app close for privacy.
