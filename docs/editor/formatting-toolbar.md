# Formatting Toolbar

The formatting toolbar provides one-click access to common Markdown actions. It appears below the document title and above the editor for Markdown notes in Live and Source modes.

The toolbar is hidden in View mode, Zen mode, and board notes.

## Actions

| Action | Result |
| --- | --- |
| Bold | Wraps the selection in `**...**`, or inserts an empty bold pair. |
| Italic | Wraps the selection in `*...*`, or inserts an empty italic pair. |
| Strikethrough | Wraps the selection in `~~...~~`, or inserts an empty strikethrough pair. |
| Inline code | Wraps the selection in backticks, or inserts an empty backtick pair. |
| Link | Inserts Markdown link syntax, using the selection as link text when present. |
| Image | Inserts Markdown image syntax, using the selection as alt text when present. |
| Quote | Inserts a blockquote prefix or quotes the selected text. |
| Code block | Inserts a fenced code block or wraps the selection in one. |
| Task | Inserts an unchecked task item. |
| Table | Inserts a three-column Markdown table. |
| Rule | Inserts a horizontal rule on its own line. |

Bold, italic, strikethrough, and inline code toggle off when the selection is already wrapped in the same delimiters.

## Keyboard Access

Toolbar buttons can be reached with `Tab` and activated with `Enter` or `Space`. Hover a button to see its action and any available shortcut. The toolbar keeps focus in the editor after an action, so you can immediately continue typing.
